# 前端项目模板
##Pakage管理工具
```bash
- yarn
```
## 已添加/实现的功能
```bash
- umi - https://umijs.org/zh-CN/docs
- dva(二级封装的redux) - https://umijs.org/zh-CN/plugins/plugin-dva
- ant design - https://ant.design/index-cn
- typescript - https://www.typescriptlang.org/
- tslint - https://palantir.github.io/tslint/
- mock - https://umijs.org/zh-CN/config#mock
- less(已预防class名污染) - http://lesscss.cn/
- 按需加载 - https://umijs.org/zh-CN/config#dynamicimport
- echarts(echarts-for-react) - https://github.com/hustcc/echarts-for-react
```
##可能需要的插件
```bash
- 国际化适配(react-intl-universal) - https://github.com/alibaba/react-intl-universal
```
## 注意和建议
```bash
- 字体文件建议手动集中传到cdn，再使用URL的方式引入。由于字体文件一般比较大，这样做可以所有项目共享字体，也可以做到多项目之间不上传重复的字体文件，相当于控制了项目的体积。
```
